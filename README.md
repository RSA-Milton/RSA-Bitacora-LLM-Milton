# RSA-Bitacora-LLM-Milton

Bitácora personal de sesiones de programación asistida con LLM de Milton (Red Sísmica del Austro).

Cada archivo es un registro técnico detallado generado automáticamente al finalizar una sesión de trabajo con un agente IA (Antigravity, Claude Code, u otros).

## Estructura

```text
sesiones/
└── YYYY/
    └── MM/
        └── YYYY-MM-DD_tema_de_la_sesion.md
```

## Formato de cada sesión

Cada archivo contiene:
- **Front matter YAML**: fecha, temas, entorno, autor.
- **Hitos de la jornada**: resumen técnico de lo trabajado.
- **Decisiones y cambios**: decisiones de arquitectura o configuración tomadas.
- **Scripts/Comandos relevantes**: bloques de código clave.

## Parte del Exocortex RSA

Este repositorio es parte del sistema de memoria extendida (exocortex) de la RSA:

- **RSA-Agent-Toolkit**: Skills y reglas del agente IA.
- **RSA-Metodologias**: Índice temático federado, ADRs y guías institucionales.
- **RSA-Bitacora-LLM-{Nombre}**: Bitácoras personales de cada miembro.
